# Solodent
🏡 혼자 사는 대학생을 위한 커뮤니티, 솔로던트 협업페이지입니다
<br><br>

## Team 서생활
> 자취하는 대학생을 위한 커뮤니티 웹사이트 솔로던트 프로젝트 개발

## Project Info
- 프로젝트 기간 : 2022.10.28~2022.12.02 (4주)
- 프로젝트 인원 : 4명
    - 이봉은(팀장)
    - 김기재
    - 이동준
    - 이두용

## 기획 의도
대학생 1인 가구를 대상으로 자취에 필요한 팁을 얻거나, 중고거래, 지역 단위 모임을
진행할 수 있는 플랫폼 <솔로던트>
